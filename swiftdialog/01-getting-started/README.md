# Getting Started with swiftDialog

These scripts cover the fundamentals of swiftDialog: launching your first dialog, working with different icon types, and handling button exit codes. Each script is self-contained and can be run directly from the terminal. Start with the hello world example and work through them in order to build a solid foundation before moving on to branding and forms.

| Script | Description |
|--------|-------------|
| [hello-world.sh](hello-world.sh) | Simplest dialog — title, message, OK button |
| [basic-icon.sh](basic-icon.sh) | All icon input types — file path, app bundle, SF Symbol, built-in keyword |
| [buttons-and-exit-codes.sh](buttons-and-exit-codes.sh) | Button configuration with exit code handling |
