# 02 — Branding

Brand your swiftDialog prompts with banner images, custom fonts, hex colours, and background watermarks. These scripts show how to build polished, on-brand dialogs without any external dependencies beyond swiftDialog itself.

| Script | Description |
|--------|-------------|
| `solid-color-banner.sh` | Branded dialog using a hex color banner -- no image files needed |
| `image-banner.sh` | Photo/graphic banner with white overlay text and shadow |
| `tall-kiosk-banner.sh` | 250pt enrollment-style banner with form fields |
| `mini-notification.sh` | Compact alert using `--small` with color banner |
| `no-banner-branded.sh` | Branded dialog without a banner -- icon, fonts, and background watermark |

Full guide section: https://fullmetalmac.com/guides/swiftdialog/#banner-images-and-branding
