# 05 — Progress Bars and Timers

swiftDialog provides several ways to show progress — from simple static bars to fully dynamic command-file-driven workflows. You can update progress text, cycle through list items with live status indicators (wait, pending, progress, success, fail, error), and use countdown timers to auto-dismiss dialogs.

## Scripts

| Script | Description |
|--------|-------------|
| basic-progress.sh | Static progress bar with numbered steps |
| command-file-demo.sh | Dynamic updates via /var/tmp/dialog.log command file |
| multi-step-checklist.sh | List items with live status indicators (wait, progress, success, fail) |
| timer-examples.sh | Countdown timer with and without visible timer bar |
