# swiftDialog Examples

[swiftDialog](https://github.com/swiftDialog/swiftDialog) is an open-source macOS dialog utility created by Bart Reardon. Built with SwiftUI, it gives Mac admins a modern, scriptable way to present dialog windows, progress bars, forms, and branded prompts to end users. It replaces legacy tools like jamfHelper and cocoaDialog with a single, actively maintained binary.

## Prerequisites

- macOS 12 Monterey or later
- Install via Homebrew or the official PKG:

```bash
brew install dialog
```

Or download the latest PKG from [swiftDialog releases](https://github.com/swiftDialog/swiftDialog/releases).

## Quick Start

```bash
dialog --title "Hello" --message "Welcome to your new Mac!"
```
