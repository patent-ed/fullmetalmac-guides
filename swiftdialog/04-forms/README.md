# 04 — Forms in swiftDialog

swiftDialog includes a full set of form controls — text fields, checkboxes, dropdowns, and radio buttons — that let Mac admins collect structured input directly from users. Combined with `--json` output, form data can be parsed and fed into Jamf Pro, MDM workflows, or any shell script.

## Scripts

| Script | Description |
|--------|-------------|
| text-fields.sh | Text input with required fields, regex validation, placeholders, and secure (password) fields |
| checkboxes.sh | Checkbox options with required acceptance |
| dropdowns-and-radio.sh | Dropdown menus and radio button selections |
| json-output.sh | Complete form with --json output for scripting |
