# 06 — Workflows

Real-world Mac admin workflow patterns that combine multiple swiftDialog features — banners, progress bars, list items, forms, and dynamic updates — into complete, production-ready scripts.

## Scripts

| Script | Description |
|--------|-------------|
| jamf-enrollment.sh | Zero-touch enrollment workflow with branded banner, progress steps, and dynamic updates |
| filevault-prompt.sh | FileVault enablement dialog with security icon and deferral option |
| update-notification.sh | macOS update nudge with timer, deferral, and link to Software Update |
| software-install.sh | App installation tracker with per-app status indicators |
| data-collection.sh | User onboarding form capturing structured JSON for Jamf/MDM integration |
